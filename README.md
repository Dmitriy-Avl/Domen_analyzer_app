# 🔍 Анализатор WHOIS-данных домена

Приложение для анализа WHOIS-данных домена. Получает публичную информацию о любом домене: владельца, регистратора, даты регистрации, DNS-серверы и статусы.

## 📋 Возможности

- ✅ Получение WHOIS-данных по доменному имени
- ✅ Извлечение 9 ключевых полей информации:
  - Доменное имя
  - Владелец/организация
  - Регистратор
  - Дата регистрации
  - Дата истечения
  - Дата последнего обновления
  - DNS-серверы
  - Статусы домена
  - Страна регистранта
- ✅ Красивый форматированный вывод в терминал
- ✅ Автоматическая проверка истечения срока регистрации
- ✅ Интерактивный режим с возможностью анализа нескольких доменов
- ✅ Подробное логирование процесса

## 🔧 Модули приложения
### main.py
- Точка входа в приложение
- Запрашивает домен у пользователя
- Координирует работу всех модулей
- Обрабатывает ошибки и повторные запросы

### api_client.py
- Отвечает за получение WHOIS-данных
- Использует библиотеку python-whois
- Очищает и валидирует доменное имя
- Возвращает сырые данные

### data_processor.py
- Обрабатывает сырые WHOIS-данные
- Извлекает 9 необходимых полей
- Нормализует форматы дат
- Обрабатывает различные варианты входных данных

### report.py
- Формирует красивый вывод в консоль
- Добавляет эмодзи и форматирование
- Генерирует предупреждения о проблемах
- Проверяет истечение срока регистрации
